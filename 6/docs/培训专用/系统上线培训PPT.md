## Slide 2：为什么要上这套系统？（5 分钟）

**南宁家政行业三大痛点：**

1. 定价靠感觉 → 报价忽高忽低，有时亏了都不知道
2. 旺季忘了涨价 → 春节/三月三还按平日价做，白白少赚
3. 销售议价无底线 → 客户说"便宜点"就给打折，不知道最低能给多少

**这套系统解决什么？**

- 成本算清楚 → 每一单都知道最低多少钱才不亏
- 市场看清楚 → 知道同行卖多少钱，不会贵到没人买
- 智能调价 → 旺季自动涨、淡季稳得住、老客户有优惠

## Slide 3：系统的"三层定价"逻辑（10 分钟）⭐ 重点

```
┌──────────────────────────────┐
│  最终价格 =                    │
│  成本底线（不亏钱）            │  ← 硬约束
│  + 市场行情（有人买）          │  ← 软约束
│  + 经营策略（多赚钱）          │  ← 智能调价
└──────────────────────────────┘
```

**第一层：成本底线**
- 师傅工资 + 社保 + 耗材 + 交通 + 管理费 + 风险金
- 任何人、任何情况，不能低于这个价 × 1.1

**第二层：市场行情**
- 系统自动参考南宁本地竞品价格
- 不会贵到超出市场承受力

**第三层：经营策略**
- AI 模型自动判断：旺季加多少、该推金牌还是新手、要不要给老客户优惠

## Slide 4：系统有哪些功能？（5 分钟）

**5 个功能页面：**

1. **智能定价** → 输入需求，一键出价（销售最常用）
2. **成本查询** → 看成本明细，回答"为什么这个价"
3. **订单复盘** → 看历史订单赚了还是亏了
4. **参数设置** → 调成本和规则（运营和经理用）
5. **数据看板** → 看公司经营全局（管理层用）

**两个入口：**
- 销售用手机微信小程序
- 运营/管理用电脑浏览器

## Slide 5：销售端演示 — 怎么报一个价？（10 分钟）

**现场演示：完整报一个价（30 秒）**

1. 打开小程序 → 智能定价
2. 选：日常保洁 / 青秀 / 3室 / 熟练 / 3小时 / 工作日
3. 点"计算价格"
4. 看到结果：¥117.00，毛利率 3.6%
5. 跟客户报价 ¥117

**然后：**
- 展示"加价原因"，向客户解释为什么这个价格
- 生成报价单，微信发给客户
- 模拟客户议价：9.5 折 → 系统校验通过

| 优惠幅度 | 你能直接给吗？ |
|----------|---------------|
| 原价 | ✅ 直接报 |
| 新客/老客 9.5 折 | ✅ 系统自动通过 |
| 团购/会员 9.0 折 | ✅ 系统自动通过 |

## Slide 7：运营端演示 — 怎么调参数？（5 分钟）

**演示：调整师傅时薪**

1. 打开后台 → 参数设置

## Slide 9：常见问题 FAQ（5 分钟）

**Q：系统报价比老定价高，客户跑了怎么办？**
A：如果老定价是亏钱的，那跑了的客户本来就在亏钱做。用系统定价后虽然丢了一些低价单，但留下来的单都赚钱。

**Q：旺季价格翻倍，客户能接受吗？**
A：全南宁的同行旺季都涨价。我们的涨幅是基于市场数据算出来的，不会比同行离谱。

**Q：客户说美团更便宜，怎么办？**
A：先确认美团报价包含的项目和时长是否一致。很多时候美团标的是"2小时基础价"。确认后可使用系统优惠权限。

**Q：小程序卡了怎么办？**
A：刷新微信或重启小程序。还有电脑浏览器版作为备选。
