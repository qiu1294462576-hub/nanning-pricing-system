## 一、成本模型方法论

### 1.1 四级成本分摊模型

采用自下而上的成本核算方法，将单次服务成本分解为四个层级：

```
总成本 = 直接成本 + 间接成本 + 风险成本
保本价 = 总成本（管理费已含）
目标售价 = 保本价 / (1 - 目标毛利率)
```

#### 第一级：直接成本

| 成本项 | 计算方式 | 参数来源 |
|--------|---------|---------|
| 人力成本 | 师傅到手时薪 × 社保系数 × 服务时长 | cost_parameters.csv |
| 耗材成本 | 基础耗材费率 × 户型系数（或面积/台数） | cost_parameters.csv |
| 交通成本 | 按城区固定加价 | cost_parameters.csv |

#### 第二级：间接成本（管理费）

按订单金额的 6% 计提，包含平台运营、客服、保险等管理开支。

由于管理费基于最终订单金额计算（存在循环依赖），实际计算公式为：

```
管理前成本 = 直接成本 + 风险成本
保本价 = 管理前成本 / (1 - 管理费率)
管理费 = 保本价 × 管理费率
```

#### 第三级：风险成本

按直接成本的 10% 计提，用于补偿：
- 工时超时偏差（实际服务时间超过预估）
- 耗材额外消耗
- 交通意外绕路

#### 第四级：利润预留

目标毛利率 25%~35%，覆盖企业运营利润、研发投入、市场推广等。

### 1.2 与旧模型的关键区别

|------|-------------------|-------------------|
| 计算方法 | `标准价格 × 0.85`（售价倒推） | 四级成本正向求和 |
| 成本结构 | 未使用 config.yaml 的 cost_structure | 完整使用所有参数 |

### 1.3 参数校准依据

所有成本参数基于南宁本地市场调研和行业报告：

- **师傅时薪**：新手 20元/h、熟练 28元/h、金牌 38元/h（参考美团/58到家平台抽成后的师傅到手收入）
- **社保系数**：1.20（雇主养老保险 16% + 医疗保险 6.8% + 失业 0.5% + 工伤 0.2% ≈ 23.5%，取整按 20% 计）
- **耗材费率**：区分四种服务类型，参考行业平均物料消耗
- **交通加价**：按城区距市中心距离分级，青秀 5元 ~ 武鸣 25元
- **时间段加价**：工作日 1.0x（基准）、周末 1.1x、节假日 1.2x、旺季 1.3x（春节/三月三/回南天），直接乘算人力成本

| 服务类型 | 亏损订单数 | 亏损总额 |
|---------|-----------|---------|
| 开荒保洁 | 789 | 64,709 元 |

| 深度清洁 | 97 | 4,670 元 |
| 家电清洗 | 63 | 1,120 元 |

**分析**：开荒保洁是亏损重灾区（占亏损总额的 68.5%）。原因分析：
1. 开荒保洁按面积定价，但实际工时往往超出预估（旧房开荒难度大）
2. 交通成本占比高（开荒通常需要携带大量设备）
3. 客户对开荒保洁价格敏感，竞争激烈

### 3.4 亏损原因总结

| 亏损原因 | 影响程度 | 说明 |
|---------|---------|------|
| 工时偏差 | ★★★★★ | 实际服务时间超过基准，人力成本上升 |
| 交通成本过高 | ★★★★ | 远郊城区（邕宁、武鸣）亏损率更高 |
| 报价过低 | ★★★ | 部分订单接近或低于保本价接单 |
| 耗材超标 | ★★ | 开荒保洁耗材用量超预期 |

### 3.5 与 Phase 2 发现的交叉验证

Phase 2 发现回南天期间深度清洁均价异常偏低（-54.7%）。通过新成本模型复盘：

- 回南天深度清洁订单的实际成本并未降低
- 但订单均价下降 54.7%，导致大量深度清洁订单在此期间亏损
- **结论**：回南天期间应强制执行保本价下限，避免为抢占市场而亏本接单

5. **金牌师傅聚焦高价值订单**：金牌师傅应优先分配给高客单价的家电清洗和深度清洁
