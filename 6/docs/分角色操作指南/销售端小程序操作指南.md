## 二、智能报价：一键获取价格

### 2.1 标准报价流程（30 秒完成）

**Step 1**：进入"智能定价"页面

**Step 2**：依次选择客户需求：

| 选项 | 举例 |
|------|------|
| 服务类型 | 日常保洁 / 深度清洁 / 开荒保洁 / 家电清洗 |
| 城区 | 青秀 / 良庆 / 江南 / 西乡塘 / 兴宁 / 邕宁 / 武鸣 |
| 户型 | 1室 / 2室 / 3室 / 4室+ / 别墅 |
| 师傅等级 | 新手 / 熟练 / 金牌 |
| 服务时长 | 拖动滑块或手动输入（小时） |
| 时间段 | 工作日 / 周末 / 节假日 / 旺季 |

**Step 3**：选择定价模式：
- **AI 智能定价**（推荐）：系统自动计算最优价格，综合考虑成本、市场、供需
- **纯规则定价**：仅使用规则引擎，不含 AI 调价

**Step 4**：点击"计算价格"按钮

**Step 5**：查看结果页面：

### 2.2 如何向客户解释价格

面对客户"为什么这么贵"的询问，按以下顺序解释：

1. **先说成本**："阿姨到手时薪 × 社保 × 时长，这个是硬成本"
2. **再说市场**："我们参考了南宁同区域同服务的报价，在这个范围内"
3. **再说价值**："我们家师傅都是持证上岗，有保险，不满意可以返工"

> 💡 话术模板："姐，咱们这个价格其实是三部分构成的：第一是阿姨的工资和社保，这个是固定的；第二是我们参考了青秀这边其他平台的报价，不会比人家贵；第三是我们金牌师傅比普通阿姨做得更细致，客户满意度更高。您看这个价格可以吗？"

## 三、生成报价单

### 3.1 生成电子报价单

计算完成后，点击底部的"生成报价单"按钮：

1. 系统自动填充客户信息（可手动修改）
2. 确认报价详情无误
3. 点击"发送给客户"
4. 选择发送方式：微信转发 / 复制链接 / 保存图片

### 3.2 报价单包含内容

- 公司名称、Logo、联系方式
- 服务项目明细（类型、时长、师傅等级）
- 价格明细（各项费用分解）
- 最终报价（大写 + 小写）
- 有效期（默认 7 天，旺季 3 天）
- 备注条款（加项计费规则、取消政策）

## 四、成本查询

### 4.1 什么是成本查询？

如果你或客户想知道"这个报价里各项成本分别是多少"，使用成本查询页面：

1. 输入和智能定价相同的参数
2. 系统展示成本明细：
   - 人力成本：¥XXX（师傅时薪 × 社保 × 时长 × 时段系数）
   - 耗材成本：¥XXX
   - 交通成本：¥XXX
   - 管理成本：¥XXX
   - 风险裕量：¥XXX
   - **保本价：¥XXX**
3. 饼图展示各项成本占比

### 4.2 什么时候用？

- 客户对价格有疑问，需要透明展示成本
- 内部培训新销售，了解定价逻辑
- 议价时快速判断"最低能给多少"

## 五、议价处理

### 5.1 议价三步法

**第一步：确认客户真实诉求**

不是所有"太贵了"都是要降价，先判断：
- 预算有限 → 推荐降档（金牌→熟练 或 深度清洁→日常保洁）
- 比价 → 展示服务差异（保险、售后、师傅资质）
- 习惯性砍价 → 小额优惠或赠送下次优惠券

**第二步：在系统中输入优惠**

在报价结果页，点击"申请优惠"：
- 选择优惠类型：新客 9.5 折 / 老客户 9.5 折 / 团购 9.0 折
- 系统自动校验：**折扣后是否 ≥ 保本价 × 1.1**
  - ✅ 通过 → 生成折扣价
  - ❌ 不通过 → 系统弹窗提示"此折扣超出成本承受范围"，需主管审批

**第三步：记录议价结果**

每次议价务必在"议价原因"栏简要记录，用于后续数据分析：
- "客户对比美团报价，9.5 折成交"
- "老客户第 6 次下单，9.5 折"
- "同小区 3 户团购，9.0 折"

### 5.2 权限说明

| 你的操作 | 是否需要审批 |
|----------|-------------|
| 按系统报价成交 | 不需要 |
| 新客 9.5 折 / 老客户 9.5 折 | 不需要（系统自动通过） |
| 9.0 折（团购/会员） | 不需要（系统自动校验） |
| 低于 9.0 折 | **需要主管审批** |

## 六、常见问题

**Q1：客户要求比美团便宜，怎么办？**
> A：先确认美团上的报价是同服务、同时长的吗？很多平台报的是"2小时基础价"，实际到现场后加钟、加项。我们的报价包含 3 小时全流程。如果确实对方更低且不低于我方保本价，可匹配 9.5 折。

**Q2：客户嫌金牌师傅太贵，推新手师傅会不会出问题？**
> A：新手师傅适合标准化的日常保洁。深度清洁、开荒保洁、家电清洗建议至少熟练以上，特别是别墅和高端小区客户，推荐金牌。

**Q3：报价有效期是多久？**
> A：平时 7 天有效。旺季（春节前/三月三）3 天有效，因为价格波动大。

**Q4：加项怎么报价？**
> A：在智能定价页面选择加项类型（擦玻璃+30元、油烟机清洗按家电清洗报价），系统会自动叠加计算。

**Q5：现场发现面积或户型跟客户说的一致，要重新报价吗？**
> A：如果偏差在 10% 以内，维持原价；超过 10%，跟客户沟通后重新报价，确保不亏。
