# 保本价计算器使用培训

> 版本: 1.0 | 日期: 2026-05-22 | 适用对象: 销售团队 / 客服人员

---

## 一、计算器打开方式

### 电脑端
1. 打开文件管理器，找到 `frontend/03_break_even_calculator.html`
2. 双击文件，自动在浏览器中打开
3. 或在浏览器地址栏输入：`file:///D:/project/frontend/03_break_even_calculator.html`

### 手机端
1. 将文件通过微信/钉钉发送到手机
2. 点击文件选择「在浏览器中打开」
3. 或扫描电脑端分享的二维码（可自行生成）

> **说明**：这是个独立 HTML 文件，不需要服务器，直接打开就能用。首次打开需要网络加载 ECharts 图表库，后续可离线使用。

---

## 二、界面介绍

计算器从上到下分为四个区域：

### 顶部：服务信息输入区
在这里填写客户的基本信息。

### 中部：计算结果区
实时显示：总成本、保本价、目标售价（三大数字），以及五个成本构成的金额。

### 中下部：成本占比饼图
直观展示各成本项的占比，帮助向客户解释价格构成。

### 底部：操作按钮 + 参数调整
- **复制价格明细**：一键复制格式化报价单
- **导出 CSV**：保存为电子表格
- **调整成本参数**：高级功能，可修改内部成本参数

---

## 三、各输入字段含义

| 字段 | 说明 | 影响什么 |
|------|------|---------|
| 服务类型 | 客户需要的服务种类 | 影响耗材费率和计价方式 |
| 师傅等级 | 派单师傅的技能等级 | 影响人力成本（等级越高时薪越高） |
| 城区 | 服务所在城区 | 影响交通加价（武鸣最贵，青秀最便宜） |
| 时间段 | 工作日/周末/节假日/旺季 | 影响人力成本（周末+10%，节假日+20%，旺季+30%） |
| 服务时长 | 预计需要多少小时 | 直接影响人力成本和总价 |
| 户型 | 房屋大小 | 影响耗材用量（别墅耗材最多） |
| 目标毛利率 | 公司希望赚多少 | 越高售价越高（默认 30%） |

## 四、典型场景示例

### 场景 1：日常保洁报价（最常用）
> 客户咨询：青秀区 3 室 2 厅，日常保洁，预计 3 小时

**输入**：
- 服务类型：日常保洁
- 师傅等级：熟练
- 城区：青秀
- 时长：3 小时
- 户型：3室
- 目标毛利率：30%

**输出**：
- 保本价：129 元（43 元/小时）——绝对下限
- 目标售价：184 元（61 元/小时）——建议报价 180~200 元

**话术**：
> "青秀区日常保洁 3 小时，包含清洁剂和工具，熟练师傅上门服务，总共 180 元。这已经是我们给您的优惠价了。同样的服务在平台上要 200 元以上。"

### 场景 2：开荒保洁报价（注意亏损风险）
> 客户咨询：武鸣区 120㎡ 新房开荒，预计 6 小时

**输入**：
- 服务类型：开荒保洁
- 师傅等级：熟练
- 城区：武鸣
- 时长：6 小时
- 户型：3室

**输出**：
- 保本价：469 元——绝对下限
- 目标售价：670 元——建议报价 650~700 元

**话术**：
> "武鸣区距离市区较远，开荒保洁需要带专业设备和清洁剂。6 小时深度清洁，含交通费，总共 650 元。远郊城区交通成本高，这个价格已经扣除我们的利润了。"

### 场景 3：家电清洗（高利润业务，优先推荐）
> 客户咨询：青秀区清洗 3 台空调

**输入**：
- 服务类型：家电清洗
- 师傅等级：熟练
- 城区：青秀
- 时长：2 小时（3 台约 2 小时）
- 户型：3室

**输出**：
- 保本价：108 元
- 目标售价：154 元

**话术**：
> "3 台空调清洗，熟练师傅上门，含专用清洗剂，总共 150 元。家电清洗是我们利润最薄的服务，这个价格非常实惠了。"

### 场景 4：远郊服务（邕宁/武鸣，注意亏损）
> 客户咨询：邕宁区别墅 5 小时深度清洁

**输入**：
- 服务类型：深度清洁
- 师傅等级：金牌
- 城区：邕宁
- 时长：5 小时
- 户型：别墅

**输出**：
- 保本价：约 310 元
- 目标售价：约 440 元

**话术**：
> "邕宁区别墅深度清洁 5 小时，金牌师傅上门。邕宁距离远，交通成本高，别墅清洁耗材也多。总价 440 元，已经是保本价附近了。"

## 五、如何向客户解释保本价

客户问"为什么你们比美团贵？"时：

> "我们的报价包含很细的成本项。我给您算算：师傅的人工费占大头，社保和管理费加一起也占不少，还有交通费、清洁剂耗材。最后公司还要留一点点利润才能活下去。
>
> 美团上的价格很多是补贴价，师傅到手很少，长期服务质量没法保证。我们给您的是实打实的成本价，保本不亏就行，利润很薄的。"

## 六、价格审批规则

| 场景 | 规则 |
|------|------|
| 报价 > 目标售价 | 正常报价，无需审批 |
| 保本价 < 报价 < 目标售价 | 需要销售主管审批 |
| 报价 < 保本价 | **禁止接单**，除非总经理特批 |
| 批量订单（>10单/月） | 可在保本价上 +5% 给折扣 |

## 七、常见问题

**Q: 客户说 "太贵了" 怎么办？**
A: 打开计算器，逐项给客户看成本构成。重点说明人工成本和交通成本是刚性的。可适当降低目标毛利率（滑动条调低），但不能低于保本价。

**Q: 同样的服务，为什么武鸣比青秀贵？**
A: 武鸣交通成本 25 元，青秀只有 5 元。这个差价 20 元就是师傅往返的油费和时间的成本。

**Q: 开荒保洁为什么特别贵？**
A: 开荒保洁耗材多（3元/㎡ × 面积）、耗时长（新手慢所以成本更高）、而且按面积计费，大户型的开荒价格自然高。

**Q: 我能不能给熟客打折？**
A: 可以，但最多打到目标售价附近，不能低于保本价。例如保本价 100 元，目标售价 143 元，你可以在 120-143 之间给折扣。

**Q: 为什么周末和节假日的价格更高？**
A: 周末人力成本系数 1.1x（+10%）、节假日 1.2x（+20%）、旺季（如回南天）1.3x（+30%）。因为师傅在非工作日/旺季接单需要更高报酬。系统在计算保本价时已自动包含这部分加价，无需手动调整。

**Q: 调整成本参数会不会影响数据？**
A: 会。调整参数后所有价格实时更新。建议只有财务或运营主管才能修改默认参数。普通销售人员使用默认参数即可。
