# 数据字典

> 版本: 1.0 | 更新: 2026-05-21 | 用途: 南宁家政定价系统数据字段说明

---

## 1. 竞品价格数据 (`competitive_prices.csv`)

| 字段名 | 类型 | 长度 | 必填 | 说明 | 示例 |
|--------|------|------|------|------|------|
| 日期 | DATE | 10 | 是 | 数据采集日期 | 2026-05-21 |
| 平台 | VARCHAR | 20 | 是 | 竞品平台名称 | 美团家政 |
| 城区 | VARCHAR | 10 | 是 | 南宁7大城区 | 青秀 |
| 小区 | VARCHAR | 30 | 是 | 具体小区名称 | 荣和大地 |
| 服务类型 | VARCHAR | 10 | 是 | 家政服务品类 | 日常保洁 |
| 服务等级 | VARCHAR | 5 | 是 | 服务人员等级 | 熟练 |
| 户型 | VARCHAR | 10 | 否 | 房屋户型 | 3室2厅 |
| 建筑面积 | INT | 5 | 否 | 房屋面积（㎡） | 100 |
| 服务时长 | DECIMAL | 4,1 | 否 | 服务小时数 | 3.0 |
| 报价 | DECIMAL | 8,2 | 是 | 平台标价（元） | 120.00 |
| 促销价 | DECIMAL | 8,2 | 否 | 活动优惠价 | 98.00 |
| 报价来源 | VARCHAR | 10 | 是 | 采集方式 | 爬虫 |

| 最低起订 | VARCHAR | 最小服务时长/面积 | 3 |

## 7. 通用编码规范

### 城区编码

| 编码 | 城区 | 档次 |
|------|------|------|
| QX | 青秀 | 高端 |
| LQ | 良庆/五象 | 中高端 |
| JN | 江南 | 中端 |
| XXT | 西乡塘 | 中端 |
| XN | 兴宁 | 中端 |
| YN | 邕宁 | 中低端 |
| WM | 武鸣 | 低端 |

### 服务类型编码

| 编码 | 服务类型 |
|------|---------|
| DAILY | 日常保洁 |
| DEEP | 深度清洁 |
| FRESH | 开荒保洁 |
| APPLIANCE | 家电清洗 |

### 区域格式标准

统一格式: `南宁-{城区}-{小区名}`
示例: `南宁-青秀-荣和大地`
