## 一、看板概述

`02_analysis_dashboard.html` 是南宁家政定价系统 Phase 2 的交互式分析看板，基于 ECharts 5.4.3 构建，可在浏览器中直接打开（无需服务器）。

**打开方式**: 在浏览器地址栏输入文件路径，或直接拖拽 HTML 文件到浏览器窗口。

**依赖文件**: 看板依赖 `output/analysis_data.js`（由 `scripts/statistical_analysis.py` 生成），两个文件的相对路径关系为：
```
frontend/02_analysis_dashboard.html  ← 引用 →  ../output/analysis_data.js
```
请勿移动任一文件，否则看板将无法加载数据。

**网络要求**: 看板通过 CDN 加载 ECharts 5.4.3，首次打开需要网络连接（后续浏览器缓存可支持离线查看）。

## 三、图表说明

### 3.1 价格影响因子分析（全宽热力图）

**图表类型**: ECharts heatmap

**展示内容**: 9 个潜在影响因素（服务类型、城区、服务等级、小区档次、旺季等级、周末、三月三、回南天、春节）对价格的影响程度。

**指标解读**:
- η²效应量：衡量该因素对价格变异的解释比例（0-100%），数值越大影响越大
- 显著性标注：绿色「显著」表示 ANOVA p<0.05；灰色「不显著」表示统计上未达显著水平

**关键洞察**: 服务类型 η²=76.5%，是价格的第一驱动力；其他因素单独解释力均不足 5%。

### 3.2 南宁各城区供需热力图

**图表类型**: ECharts heatmap（7城区 × 5个月）

**展示内容**: 各城区在不同月份的供需比（预估需求量 / 预估供给量）。

**颜色含义**:
- 🟢 **绿色**（<1.0）：供过于求，师傅闲置风险
- 🟡 **黄色**（1.0-2.0）：供需平衡
- 🔴 **红色**（>3.0）：严重供不应求，需紧急调度或提价

**悬停信息**: 鼠标悬停可查看该城区×月份的供需比、样本均价、样本订单数。

### 3.3 淡旺季价格与订单量趋势

**展示内容**:

- 红色虚线：月均利润率（%）

**标注区域**:
- 🟡 黄色区域：春节前超级旺季（1月）
- 🟢 绿色区域：回南天（3-4月，深度清洁旺季）
- 🔴 文字标注：三月三（4月21日前后）

**数据周期**: 2026年1月-5月，按月聚合。

### 3.4 各城区利润率与亏损分布

**图表类型**: ECharts 柱状图 + 折线图

**展示内容**: 7 个城区的平均客单价（柱状图）和平均利润率（折线图）对比。

**用途**: 快速识别利润率偏低的城区（如邕宁 17.0%、良庆 18.7%），制定区域性调价策略。

### 3.5 服务类型 × 技能等级 利润率对比

**图表类型**: ECharts 分组柱状图

**展示内容**: 4 种服务类型 × 3 个技能等级的平均利润率对比。

**用途**: 
- 验证技能等级定价梯度是否合理（金牌应>熟练>新手）
- 识别利润率异常组合（如某服务×等级利润率明显偏低）

### 3.6 低利润订单明细表

**展示内容**: 毛利率最低的订单详情（默认显示毛利率<15%的订单）。

**列说明**:

| 列名 | 说明 |
|------|------|
| 订单ID | NN 前缀的唯一订单号 |
| 城区 | 7 个南宁城区 |
| 订单金额 | 实际收费金额（元） |
| 标准成本 | 根据基准费率估算的服务成本（元） |
| 毛利润 | 订单金额 - 标准成本（元） |
| 毛利率 | 毛利润 / 订单金额 × 100% |

**行颜色**:
- 🔴 红色：亏损（毛利率<0）
- 🟡 黄色：边缘（毛利率<10%）
- 白色：偏低（毛利率10-15%）

**排序**: 点击「毛利率」列表头可按毛利率升/降序排列。

## 四、筛选器使用

看板提供 4 个下拉筛选器，位于指标卡片下方：

| 筛选器 | 选项 | 影响范围 |
|--------|------|---------|
| 城区 | 全部 / 青秀 / 良庆 / 江南 / 西乡塘 / 兴宁 / 邕宁 / 武鸣 | 所有图表 + 明细表 + KPI |
| 服务类型 | 全部 / 日常保洁 / 深度清洁 / 开荒保洁 / 家电清洗 | 所有图表 + 明细表 + KPI |
| 月份 | 全部 / 2026-01 ~ 2026-05 | 所有图表 + 明细表 + KPI |
| 旺季 | 全部 / 超级旺季 / 普通旺季 / 平季 | 所有图表 + 明细表 + KPI |

**使用方式**:
1. 单选或多级联动：选择多个筛选器可实现交叉筛选（如「青秀 + 深度清洁 + 3月」）
2. 重置：点击「重置筛选」按钮恢复全部数据
3. 当前筛选信息：筛选栏右侧显示「当前筛选: N 条订单」

**注意**: 供需热力图和季节性趋势图使用全量月度/城区聚合数据，**不受筛选器影响**（始终展示全局视图）。

## 六、常见问题

**Q: 打开 HTML 后图表不显示？**
A: 检查 1) 网络是否可达（需要加载 ECharts CDN）；2) `output/analysis_data.js` 文件是否存在且位于正确路径；3) 浏览器控制台是否有报错信息。

**Q: 图表中文显示为乱码？**
A: 确认浏览器编码设置为 UTF-8。如仍有问题，检查 `analysis_data.js` 文件是否以 UTF-8 编码保存。

**Q: 所有订单都显示为盈利，未见亏损？**
A: 当前 200 条样本数据中，所有订单的毛利率均>0。看板使用「低利润」（毛利率<10%）作为预警阈值，明细表中显示的是利润率最低的订单。随着数据量增加和实际运营数据接入，可能出现真实亏损订单。

**Q: 如何修改低利润预警阈值？**
A: 编辑 `02_analysis_dashboard.html`，搜索 `renderLossTable` 函数中的 `r.毛利率 < 15`，将 15 修改为所需阈值。

**Q: 供需热力图显示的数据为什么和订单数据月份不完全匹配？**
A: 供需数据来自 `supply_demand_data.csv`（基于行业报告和平台数据的宏观估算），订单数据来自 `sample_orders.csv`（实际成交样本）。两者来源不同，已在图表悬停信息中同时展示以供交叉参考。
