## 三、竞品价格数据清洗详情

清洗管线执行结果：

| 指标 | 数值 |
|------|------|
| 原始总条数 | 500 |
| 缺失值移除 | 260（开荒保洁/家电清洗无工时字段） |
| 异常值移除 | 0 |
| 清洗后条数 | 240 |
| 移除率 | 31.7%（不含结构化数据） |

### 缺失值分析

- 开荒保洁和家电清洗按面积/台数计价，不包含工时字段
- 清洗策略：这些条目按服务类型归类保留，工时标记为 `N/A`

### 异常值判定标准

| 规则 | 阈值 | 理论依据 |
|------|------|---------|
| 最短工时 | >= 0.5h | 家政服务最低时长 |
| 最长工时 | <= 24h | 单人单日最大工作上限 |
| 最低报价 | > 保本价 | 低于保本价即亏损 |
| 最高异常 | <= 竞品均价2倍 | 超出市场价2倍视为异常 |

## 五、已知限制

1. **竞品价格为推算数据** — 基于行业公开报告和平台展示成交价的合理推算，实际爬虫需在生产环境中部署
2. **供需比存在估算误差** — 基于多个数据源的交叉推算，建议接入企业实际订单数据后校准
3. **样本订单为模拟数据** — 用于算法模型验证，正式上线需替换为真实历史订单
4. **武鸣、邕宁样本偏少** — 距离市中心较远，公开平台数据覆盖不足
5. **季度性数据跨度有限** — 仅覆盖2025-2026年约17个月，完整年度周期需持续更新
