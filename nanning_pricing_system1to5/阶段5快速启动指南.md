# 南宁家政动态定价系统 — 快速启动指南

## 环境要求

- Python 3.10+ (推荐 miniconda3)
- 微信开发者工具 (仅小程序需要，[下载](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html))
- 本系统已在 Windows 11 测试通过

---

## 一、安装依赖 (一次性)

```bash
cd nanning_pricing_system
pip install -r requirements.txt
```

如果 `chinese-calendar` 安装失败可以跳过，不影响核心功能。最小依赖：`fastapi uvicorn pandas numpy scikit-learn xgboost scipy pyyaml matplotlib seaborn openpyxl`

---

## 二、生成数据 + 训练模型 (一次性)

```bash
python scripts/generate_realistic_data.py   # 生成 8000 条仿真订单 → data/nanning/
python scripts/train_model.py               # 训练 XGBoost 模型 → output/models/
```

已执行过的可跳过。模型文件在 `output/models/pricing_model.json` 就表示已完成。

---

## 三、启动后端

```bash
uvicorn backend.main:app --host 0.0.0.0 --port 8000
```

看到以下输出表示成功：
```
[OK] 定价引擎初始化完成: 模型已加载
INFO: Uvicorn running on http://0.0.0.0:8000
```

验证：浏览器打开 `http://localhost:8000/`，应返回 JSON 状态信息。

---

## 四、打开前端

### 方式一：微信小程序

1. 打开 **微信开发者工具**
2. 导入项目 → 选择目录 `nanning_pricing_system/phase5/miniprogram`
3. AppID 选「游客模式」即可
4. 点击 **编译**
5. 5 个 tab 页：智能定价 / 成本查询 / 订单复盘 / 参数设置 / 数据看板

### 方式二：浏览器版 (Web SPA)

1. 确保后端已启动
2. 浏览器打开 `http://localhost:8000/app/`
3. 同样 5 个功能页面，自适应移动端和桌面端

---

## API 文档

启动后端后，打开 `http://localhost:8000/docs` 可查看所有 API 的交互式文档 (Swagger UI)。

---

## 常见问题

| 问题 | 解决 |
|------|------|
| 微信小程序连不上后端 | 检查 `phase5/miniprogram/app.js` 中 `apiBase` 是否为 `http://localhost:8000` |
| 游客模式限制提示 | 无 AppID 的正常提示，不影响功能测试 |
| 模型未加载 | 先运行 `python scripts/train_model.py` |
| 订单数据不存在 | 先运行 `python scripts/generate_realistic_data.py` |
